# Employee Management System - Frontend

The frontend of the Employee Management System is built using **React.js** and **Material UI**. 
It provides a modern and intuitive interface for managing employees, departments, and performance reviews. The frontend communicates with the backend APIs created using .NET Core Web API.

## Features

- Add, edit, view, and delete employees.
- Soft delete functionality for employees.
- Search and filter employees by name, department, position, and performance score.
- Pagination for data visualization.
- Manage departments with features like budget allocation and department hierarchy.
- Display employee performance reviews and calculate average scores per department.

## Technologies Used

- React.js
- Material UI
- Axios
- React Router
- SweetAlert2
